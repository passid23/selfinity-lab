# Selfinity-Lab
Code of the SYP Project: Selfinity-Lab, which is a Quiz that recommends appropriate Courses

# Setup

## Vorbereitung *(Einmalig oder bei Start)*

```bash
git clone <URL>
npm install  # In allen 3 Ordnern: Root, /frontend, /backend!
npm run dev  # Im Hauptordner zum Testen → auf den beiden localhost gehen
```

## Arbeiten

```bash
git checkout main
git pull origin main      # Hol dir alle neuen Updates!
git checkout -b feature/name  # Erstellt Branch

# ...Coden

git add .
git commit -m "feat: Quiz Logik"
git push origin feature/name
```
